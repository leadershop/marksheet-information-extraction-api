# Sample Marksheets Directory

Place your test marksheet files here (PDF, JPG, PNG).

## Testing Tips

1. **Diverse Formats**: Test with different board marksheets (CBSE, State boards, Universities)
2. **Quality Levels**: Try both clear scans and slightly lower quality images
3. **File Types**: Test both PDFs and images

## Where to Find Sample Marksheets

- Search for "sample marksheet PDF" online
- Use publicly available marksheet templates
- Create mock marksheets for testing

## Note

Sample files are gitignored to avoid committing large files or sensitive data.
